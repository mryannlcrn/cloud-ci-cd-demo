# Cloud CI/CD Demo

This project shows a simple CI/CD pipeline by using GItHub Actions.

# Features
- Runs build and tests whenever a code is pushed
- Automatically deploys to GitHub Pages
- Keeps secrets safe using GitHub Secrets and Variables
